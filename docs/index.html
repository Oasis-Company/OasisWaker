🌴 OasisWaker

Crowdsourced edge infrastructure. Your free-tier cloud quota, aggregated.

View on GitHub Documentation
⚠️ Honest status: v1.0 is functional but has known security issues. See CRITIQUE.md for details. v2.0 (true P2P) is not yet implemented.
🔗

Works with What You Have

Cloudflare Workers, Vercel Edge, Supabase Edge — use your existing free-tier quota. No new accounts needed beyond what you already have.

🔐

Client-Side Encryption

Data is encrypted with AES-256-GCM before it leaves your machine. Node operators cannot read what they store.

🌐

Global by Default

Edge nodes span every region where Cloudflare, Vercel, or Supabase operate. No single geographic point of failure.

💡

Vision: True P2P

v2.0 aims for a DHT-based mesh where nodes discover each other without a central coordinator. Read the vision →

Two Versions, One Project

✅ Current: v1.0

Centralized CLI Tool

  • ✅ oasiswaker init/login/deploy
  • ✅ OAuth2.0 PKCE flow
  • ✅ Deploys to Cloudflare/Vercel/Supabase
  • ✅ Unified block storage API
  • ⚠️ Central coordinator (SPOF)
  • ⚠️ Known security issues
🔮 Vision: v2.0

True P2P Mesh

  • 🔮 DHT node discovery (Kademlia)
  • 🔮 Erasure coding for data redundancy
  • 🔮 No central coordinator
  • 🔮 Incentive layer (credits)
  • 🔮 Monorepo package structure
  • ❌ Not implemented yet
